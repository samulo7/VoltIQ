## 2. 术语与基础口径

### 2.1 直接交易定义（证据ID：KB-UT-001）
在本测试口径中，“直接交易”指发电侧与用电侧（或售电公司代理）按市场规则直接成交电量与电价，由交易机构组织、结算机构统一出清与结算的市场化交易方式。  
核心要点：  
1. 交易双方按规则撮合成交。  
2. 价格由市场形成，不是单一目录电价。  
3. 仍需遵守当地交易中心与电网结算规则。

### 2.2 常见结算方式（证据ID：KB-UT-002）
常见结算口径可分为：  
1. 按月结算：按月度交易电量与价格进行统一结算。  
2. 分时结算：按峰平谷（或分时段）口径分别结算。  
3. 偏差考核后结算：先计算偏差电量，再叠加偏差费用或考核系数。  
说明：不同省份规则会有细节差异，最终以当地交易细则为准。

### 2.3 偏差考核风险解释（证据ID：KB-UT-003）
客户担心偏差考核时，可按以下逻辑解释：  
1. 偏差本质是“申报与实际用电不一致”的成本反映。  
2. 通过更准确的用电预测、分时策略与滚动修正，可以降低偏差风险。  
3. 售电侧可提供“月内跟踪 + 预警机制”，把风险前置管理，而不是事后被动承担。  
建议话术：  
“我们会把偏差管理当作持续服务，提前预警和调整，目标是把波动控制在可管理范围内。”

## 3. 客户沟通模板

### 3.1 可直接发送客户的简短话术（证据ID：KB-UT-004）
“本次方案采用市场化直接交易模式，电价与电量按交易规则形成并结算。我们会结合您的历史负荷做月内滚动跟踪，重点控制偏差考核风险，确保价格与风险都在可控区间。”

### 3.2 销售沟通注意事项（证据ID：KB-UT-005）
1. 不承诺“绝对最低价”，应强调“综合成本最优 + 风险可控”。  
2. 先解释规则再谈价格，避免客户对结算机制理解偏差。  
3. 每次沟通都给“下一步动作”（如补负荷数据、确认合同周期、安排复盘时间）。

## 5. 连续对话测试脚本（可选）
建议连续提问顺序：  
1. 先问“直接交易定义”；  
2. 再追问“常见结算方式”；  
3. 再追问“偏差风险如何解释”；  
4. 要求“总结成一段给客户的话术”；  
5. 最后追问“再给 3 条沟通注意事项并标注依据”。  
验收口径：回答非空、内容连贯、并能返回来源引用。
